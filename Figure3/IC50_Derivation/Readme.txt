IC.mat: IC50 fold-change values for each viral strain for each NRTI combinations.
IC50_Generate.m: Produces IC.mat file.
M_generate.m: Produces Mutation_Data.mat file.
Mutation_Data.mat: Includes all possible 131,072 viral strains X, the mutation matrix ML, starting indexes of the viral strains having one, two, .. etc. point mutations Q.
